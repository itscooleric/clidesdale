# CLAUDE.md — clide-sdale

## What is clide-sdale
Framework + CLI for giving AI agents SSH access to throwaway VPSes with tmux co-dev sessions. The human can always watch and intervene.

## Rules
1. **No invisible remote work** — every remote command goes through a named tmux session
2. Keep it simple — shell scripts and minimal Node.js, no heavy frameworks
3. Will be made public — keep the repo clean, no secrets, no hardcoded IPs
